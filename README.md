# Freight Management System (Odoo)

An integrated solution designed to streamline freight operations, logistics, and fleet management. This project was developed as a customized professional solution for a specialized company to enhance operational efficiency and digitize logistics workflows.

## 📌 Project Overview
This system was built to automate freight workflows, reducing reliance on manual processes and providing full visibility into all operations through a single, Odoo-based platform. It covers everything from trip scheduling to safety compliance and driver financial management.

## ✨ Key Features
* **Trip Management:** Accurate tracking of trip logs, routes, and real-time status updates.
* **Safety Checklists:** Implementation of safety protocols to ensure vehicle and driver compliance before and during trips.
* **Driver Management:** Comprehensive system for tracking driver advances, expenses, and performance.
* **WhatsApp Integration:** Automated sending of Waybills and real-time notifications directly to drivers and clients via WhatsApp.
* **Advanced Reporting:** High-quality QWeb/PDF reports for performance analysis and logistics insights.

## 🛠 Tech Stack
* **Backend:** Python & Odoo ORM (V19 compatible)
* **Frontend:** XML, JavaScript (OWL Framework)
* **Database:** PostgreSQL
* **Deployment:** Docker & Ubuntu Environment
